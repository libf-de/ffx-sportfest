0. Vorwort

FFSportfest dient der Erfassung und Auswertung der Ergebnisse des alljährlichen Sportfests des Gymnasiums Herrmann Pistor Sonneberg und soll eine verbesserte Version des Sportfest- Programs von Felix Büchner, Hannes Frank, Robert Kienel und Stefan Loidolt sein. Wesentliche Verbesserungenspunkte sind:

-Schnellere Bedienung mit Tastenkombinationen bzw. der Tastatur allgemein

-Übersichtlichere und modernere Oberfläche mit Dateneingabe im Microsoft-Excel-Stil

-Vereinfachung der Schülereingabe durch Excel-Import

-Erweiterung des Funktionsumfangs durch:

oEinstellbare Sortierung und Filterung

oUrkundendruck mit Word-Vorlage

oErgebnislistendruck

oKranke/nicht anwesende Schüler überspringen

-Verbesserung der Arbeit an mehreren Computern gleichzeitig durch mobile Datenbanken

-Zukunftssicher durch öffentlich zugänglichen Quellcode

-Absturzsicherung

Nahezu alle Menüpunkte im Programm sind auch durch allgemeine Tastenkombinationen aufrufbar, hierbei ist es egal ob der Menüpunkt mit der Maus angewählt wird oder die Tastenkombination gedrückt wird – die Tastenkombination ist lediglich in der Regel schneller. Die Tastenkombinationen stehen rechts in den zugehörigen Menüpunkten und im Handbuch.

Im alten Programm wurde für jede Klasse eine separate Datenbank auf jedem Computer verwendet

–dies ist nun nicht mehr nötig (ich rate davon ab um die Übersicht zu behalten). Alle Schüler aller Klassen sollten in einer Datenbank erfasst werden und die Funktion „Zusammenarbeit“ mit einem USB-Stick verwendet werden.

1. Das Hauptfenster – Eingabe von Ergebnissen

1.1 Datenbank öffnen

Im Hauptfenster können Sie die von den Schülern erzielten Ergebnisse eintragen und die daraus resultierenden Punktwerte und Noten einsehen. Dazu muss zuerst eine Schülerdatenbank geöffnet werden – dies ist auf 2 Wegen möglich:

Öffnen Sie FFSportfest und klicken Sie im Hauptfenster im Menü „Datei“ auf „Datenbank öffnen“ – alternativ kann auch die Tastenkombination Strg + O verwendet werden – und wählen Sie die gewünschte Datenbank aus.

Öffnen Sie den Explorer („Dieser PC“) und navigieren Sie zur gewünschten Datenbank – die Datenbank kann mit einem Doppelklick geöffnet werden.

Sollten Sie immer die gleiche Datenbank verwenden kann in den Einstellungen die Option „Letzte Datenbank beim Start laden“ aktiviert werden – dann müssen Sie dies nicht jedes Mal selbst tun.

1.2 Dateneingabe

Anschließend wird eine Tabelle im Hauptfenster angezeigt – hier werden die Daten eingetragen. Die Spalten Name, Vorname, Geschlecht, Klasse sowie Punkte und Note können hier nicht verändert werden, dies kann nur im Datenbankeditor erledigt werden. Falls sie nur eine bestimmte Klasse/Klassenstufe und/oder Geschlecht bearbeiten wollen können Sie oben in der rechten Seitenleiste die angezeigten Daten filtern.

Um Ergebnisse einzugeben klicken Sie in die gewünschte Tabellenzelle. Sie können folgende Tasten verwenden um in der Tabelle zu navigieren:

1.3 Datenbank speichern / unter neuem Namen speichern

Wenn Daten geändert wurden muss die Datenbank gespeichert werden – Sie können dies im Menü Datei mit der Option Speichern – oder mit der Tastenkombination Strg+ S – tun. Wenn Sie die Datei unter einem anderen Namen oder an einem anderen Ort speichern möchten verwenden Sie die Option Speichern unter (ebenfalls im Datei-Menü) – bzw. die Tastenkombination Strg + Umschalt + S. Dass die Datenbank bearbeitet wurde wird durch ein Sternchen (*) im Fenstertitel angezeigt:

Wenn Sie versuchen dass Programm zu schließen und nicht gespeichert haben erscheint eine Abfrage ob Sie die Datenbank abspeichern möchten (Speichern), die Änderungen verwerfen wollen (Verwerfen) oder das Programm nicht schließen wollen (Abbrechen)

1.4 Programm beenden

Um das Programm zu beenden verwenden Sie das × oben rechts, den Menüpunkt Beenden im Menü Datei oder die Tastenkombination Strg + Q.

2. Schüler erstellen, bearbeiten, aus Excel übernehmen

Um Schüler zu erstellen, bearbeiten, etc. müssen Sie den Datenbankeditor verwenden; öffnen Sie diesen mit dem Menüpunkt Datenbankeditor im Menü Datei des Hauptfensters oder der Tastenkombination Strg + E. Darauf öffnet sich ein neues Fenster.

Ist im Hauptfenster eine Datenbank geöffnet wird diese dort automatisch geschlossen und im Datenbankeditor geöffnet. Sie sollten eine Datenbank nie gleichzeitig im Datenbankeditor und im Hauptfenster geöffnet haben da sich beide beim Speichern „ins Gehege kommen“.

2.1 Öffnen, Speichern und unter neuem Namen speichern

Öffnen, Speichern und unter neuem Namen speichern funktioniert hier genau so wie im Hauptfenster mit den Menüpunkten „Öffnen“ (Strg + O), „Speichern“ (Strg + S) bzw. „Speichern unter“ (Strg + Umschalt + S) im Menü Datei.

2.2 Neue Datenbank erstellen

Wenn Sie eine neue Datenbank erstellen möchten verwenden Sie „Neue Datenbank“, ebenfalls im Menü Datei – bzw. mit der Tastenkombination Strg+ N. Daraufhin wird eine leere Datenbank mit einem Schüler erstellt.

2.3 Datenbank mithilfe einer Excel-Tabelle erstellen

Um das Erstellen der Datenbanken zu vereinfachen kann das Programm Schülerlisten in Form von Excel-Tabellen importieren. Die Excel-Tabelle muss mindestens 2 Spalten (Name und Vorname) beinhalten. Klasse und Geschlecht können entweder aus Tabellenspalten entnommen werden oder für alle Schüler manuell eingegeben werden (falls diese nicht vorhanden sind). Es wird empfohlen, die Tabelle ebenfalls in Excel zu öffnen um die Spaltenbezeichnungen herauszufinden.

Öffnen Sie zuerst das Excel-Import-Fenster mit dem Menüpunkt „Datenbank aus Excel-Tabelle erstellen“ (Tastenkombination Strg + Umschalt + N) – ein neues Fenster öffnet sich. Folgen Sie den nummerierten Schritten:

Nachdem die Datenbank gespeichert wurde und der Import abgeschlossen ist wird die erstellte Datenbank automatisch im Datenbankeditor geöffnet, hier können falls nötig weitere Korrekturen vorgenommen werden.

2.4 Schüler hinzufügen

Um einen neuen Schüler hinzuzufügen klicken Sie in der Werkzeugleiste auf. Alternativ können Sie, wenn die letzte Zeile ausgewählt ist, 2x Enter drücken. In beiden Fällen wird eine leere Zeile am Ende angefügt und rot hervorgehoben.

2x Enter

Tragen Sie den Vor- und Nachnamen, die Klasse und das Geschlecht ein – anschließend sollte die Zeile nicht mehr hervorgehoben sein.

2.5 Schüler entfernen

Um einen Schüler zu löschen wählen Sie den gewünschten Schüler aus und klicken Sie in der Werkzeugleiste

auf(oder drücken Sie Entf auf der Tastatur). Anschließend erscheint ein Dialogfeld in welchem die Details des Schülers welcher gelöscht wird erneut angezeigt wird. Um den Schüler nun zu löschen bestätigen Sie die Abfrage mit einem Klick auf Ja.

2.6 Klassen löschen

Um eine ganze Klasse zu löschen, wählen Sie die Funktionaus der Werkzeugleiste. Daraufhin öffnet sich ein Dialog in welchem Sie eine Klasse oder Klassenstufe aus der Datenbank zum Löschen auswählen können. Wenn Sie die gewünschte Klasse ausgewählt haben müssen Sie die Sicherheitsabfrage anhaken – daraufhin wird der Knopf Löschen aktiviert.

2.7 Klassen verschieben

Um die bestehende Datenbank einer Klasse ins nächste (höhere) Schuljahr zu übertragen, gibt es die

Funktion in der Werkzeugleiste. Hierbei werden alle Schüler des Schuljahres eine Klassenstufe höher verschoben, daher erfolgt bei Aufruf dieser Funktion neben einer kurzen Erklärung derselben eine Sicherheitsabfrage.

2.8 Sortieren

Ist die Option Daten sofort Sortieren in den Einstellungen deaktiviert können mit dieser Funktion die Schüler in der Datenbank nach der Standard-Sortierung (siehe Einstellungen) sortiert werden.

3. Auswertung

Im Auswertungsfenster können die erzielten Ergebnisse ausgewertet, die Sieger ermittelt sowie Ergebnistabellen und Urkunden gedruckt werden. Um das Auswertungsfenster zu öffnen klicken Sie im Hauptfenster unten in der rechten Seitenleiste auf Auswertung öffnen. Mit den Feldern Filter und Sortierung können die gewünschten Kriterien ausgewählt werden, z.B. ist es möglich, alle Jungen einer Klassenstufe anzuzeigen und nach Punkten zu sortieren.

3.1 Daten aktualisieren

Normalerweise erfolgt die Aktualisierung der Datensätze aus dem Hauptfenster automatisch (sobald dort etwas verändert wurde), dies kann jedoch zur Sicherheit mit dieser Funktion erzwungen werden.

3.2 Neu berechnen

Normalerweise berechnet das Programm die Punkte- und Notenwerte automatisch, dies kann jedoch zur Sicherheit mit dieser Funktion erzwungen werden.

3.3 Plätze neu ermitteln

Normalerweise werden die Plätze automatisch ermittelt (also die Schüler nach den ausgewählten Kriterien sortiert), dies kann jedoch zur Sicherheit mit dieser Funktion erzwungen werden.

3.4 Druckvorschau / Drucken

Die erzielten Ergebnisse können zur späteren Auswertung in Tabellenform ausgedruckt werden. Hierfür können die Funktionen Druckvorschau sowie Drucken verwendet werden. Beim Ausdruck werden die ausgewählten Filter- und Sortierkriterien übernommen. Nach einem Klick auf Drucken öffnet sich ein Dialogfenster in welchem Sie ihren Drucker, sowie weitere Druckoptionen wie Anzahl der Kopien, Papierformat, etc., auswählen können. Sind alle Einstellungen korrekt (dies geschieht in der Regel automatisch) kann mit einem Klick auf Drucken im Dialogfeld der Druck gestartet werden.

3.5 Urkunden erstellen

Das Programm ermöglicht es ebenfalls automatisch Urkunden gemäß einer Wordvorlage zu erstellen (auszufüllen). Hierzu muss zunächst eine Vorlage in Word erstellt werden und diese in den Einstellungen ausgewählt werden (siehe 6. Urkundenvorlagen erstellen). Ist dies (schon) geschehen können Sie fortfahren; Klicken Sie auf Urkunden erstellen. Darauf werden Sie nach einem Speicherort und einem Dateinamen für die ausgefüllten Urkunden gefragt. Haben Sie die Auswahl mit Speichern bestätigt werden die erstellten Urkunden anschließend automatisch in Word (o.ä.) geöffnet von wo diese gedruckt werden können.

4. Mobile Datenbanken / Zusammenarbeit

Zur Eingabe von Ergebnissen an mehreren Computern gibt es mobile Datenbanken – sie enthalten nur eine Klassenstufe und werden nach Ergebniseingabe wieder in die Hauptdatenbank eingefügt um alle Ergebnisse zentral an einem Ort gespeichert zu haben. Um diese Funktion zu nutzen muss FFSportfest auf allen Computern installiert sein; außerdem wird ein USB-Stick o.ä. benötigt.

4.1 Begriffe

Schülerdatenbank

Datenbank welche alle Klassen & Schüler enthält

Mobile Datenbank

Enthält nur eine Klassenstufe und deren Schüler

FMD

= FFSportfest Mobile Datenbank, siehe Mobile Datenbank

Im folgenden wird der Computer, auf welchem sich die vollständige Datenbank befindet, als Hauptrechner bezeichnet; alle anderen Computer werden als Klient-Rechner bezeichnet.

4.2 Mobile Datenbank erstellen

Folgendes muss auf dem Hauptrechner erledigt werden:

Stecken Sie ihren USB-Stick an den Hauptrechner und öffnen Sie die Schülerdatenbank wie üblich im Hauptfenster. Klicken Sie anschließend in der Menüleiste im Menü Zusammenarbeit auf die Funktion FMD exportieren. Es öffnet sich ein Dialogfeld, in welchem Sie die Klassenstufe auswählen können, welche in die mobile Datenbank soll. Klicken Sie dann auf Exportieren, wählen Sie ihren USB-Stick aus und geben Sie einen Dateinamen wie „Schüler_Klasse5“ ein.

4.3 Mobile Datenbank bearbeiten

Folgendes muss jeweils auf jedem Klient-Rechner erledigt werden:

Stecken Sie ihren USB-Stick an den Klient-Rechner und kopieren Sie die mobile Datenbank auf diesen Computer, z. B. auf den Desktop (nicht erforderlich wenn für jeden Klient-Rechner ein separater USB-Stick verwendet wird welcher an diesem verbleibt). Öffnen Sie die mobile Datenbank, indem Sie doppelt darauf klicken – daraufhin wird FFSportfest im „mobilen Modus“ geöffnet (Datenbankeditor, etc. sind nicht verfügbar). Sie können dann wie gewohnt die Datenbank bearbeiten und speichern.
